# Konzept

Wenn eine Kette A eine neue Kette B erstellt , ist Chain A die Eltern-Kette der Kind-Kette B. Das System hat ursprünglich nur eine Kette. A hat keine Elternkette, die anderen haben allerdings Elternketten. Jede Kette kann 2 Kind-Ketten erstellen, welche rechte und linke Kind-Kette genannt werden.
